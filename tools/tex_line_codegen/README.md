# יצירת קוד לשורת Tex (עברית + `$...$`)

דף **סטטי** (ללא שרת): פותחים את `index.html` בדפדפן (או דרך שרת קבצים מקומי).

## לשוניות

- **יוצר שורות** — טקסט עברית + `$...$`, צבעים, משכים, `Tex`, `SmartHebWrite`, אופציה לקו תחתון לכל השורה (`Line` אחרי הכתיבה), וכו׳.
- **צירים וגרפים** — יוצר קוד ל־`Axes` או `NumberPlane` (טווחי `x_range` / `y_range` בפורמט `min, max, step`), אורכי ציר, `include_tip`, אופציה ל־`get_hebrew_template()`. **תזמון:** משך ניגון והמתנה אחרי הצירים, ולכל אובייקט — משך ניגון והמתנה אחרי (ברירות מחדל; `0` בהמתנה = בלי `wait`). **מקבילי:** «נגן במקביל לשורה הקודמת» — כמה `Create` ב־`self.play` אחת; `run_time` = מקסימום בקבוצה, `wait` אחרי הקבוצה = לפי השורה האחרונה. **מעל** כפתור **צור קוד**: אובייקטים — גרף (`plot`) או נקודה (`Dot`), צבע; ליד הכותרת **?** — דוגמאות `numpy`. **+** / **−**. `import numpy as np` רק כשיש גרף. שלושה בלוקי העתקה.

## שימוש (לשונית «יוצר שורות»)

1. **שינוי גופן** — בראש הדף: **שם גופן** ל־`get_hebrew_template(...)` (ברירת מחדל **`David Libre`**, כמו ב־`hebrew_utils`; במחשבים בלי הגופן — גופן מותקן אחר, למשל `Noto Sans Hebrew`). בקובץ המלא יתווסף ארגומנט שם רק כשהערך שונה מ־David Libre. **גודל גופן** לשורה שלמה (`font_size` ל־`Tex`): ברירת מחדל **36**; בקוד יתווסף `font_size=…` רק כשהערך שונה מ־36.
2. מזינים קטעים לפי סדר (כל תיבה = קטע נפרד; כפתור **+** מוסיף שורה).
3. בתוך קטע אפשר לשלב LaTeX עם `$...$` — הכלי מפצל **אוטומטית** לכמה ארגומנטים ל־`Tex`, כמו בפרויקט (כל בלוק `$...$` הוא מחרוזת משלו).
4. ליד כל שורה: כפתורי **B / I / U** (מודגש, נטוי, קו תחתון) — נטמעים ב־LaTeX (`\textbf`, `\textit`, `\underline` לטקסט; `\mathbf` / `\mathit` / `\boldsymbol` / `\underline` לנוסחה). חלים על כל הקטעים מאותה שורה אחרי הפיצול.
5. (אופציונלי) מסמנים **צבע שונה לכל שורה** — לכל שורה: **לוח צבעים קבוע** (לחיצה מהירה, אותו צבע בכמה שורות), שדה **Hex** (`#RRGGBB`), ובוחר צבע מובנה. הקוד המיוצר כולל קריאות ל־`set_tex_segment_color` עם מחרוזות hex.
6. (אופציונלי) מסמנים **משך זמן שונה לכל קטע (שניות)** — לכל **ארגומנט סופי** ב־`tex_parts` (אחרי פיצול `$...$`) מופיע שדה מספר; הקוד משתמש ב־`smart_heb_write_segment(..., run_time=...)` בלולאה. ברירת מחדל `1.0` שניות לכל קטע. ללא סימון: אנימציה אחת עם `SmartHebWrite`.
7. (אופציונלי) **קו תחתון לכל השורה** — אחרי סיום אנימציית הכתיבה, הקוד מוסיף `Line` מ־`DR` ל־`DL` (ימין → שמאל) מעט מתחת ל־`Tex`, ו־`Create` עם `color="#…"` (לוח צבעים + Hex). בקובץ המלא מתווספים יבואים: `Line`, `Create`, `DOWN`, `DR`, `DL`.
8. **צור קוד** — בלוקים להעתקה:
   - **בלוק 1:** `segment_run_times` (אם סומן משך לכל קטע) + `tex_parts` + `Tex` (`font_size` אם שונה מ־36) + צבעים אם סומן + הערה/קוד לקו תחתון לשורה אם סומן — להדבקה בתוך `construct` קיים.
   - **בלוק 2:** **קובץ מלא** — `from manim import Scene, Tex, config` (ו־`Line`, `Create`, `DOWN`, `DR`, `DL` אם קו תחתון), `hebrew_utils`, `config.tex_template`, מחלקה `TexLineScene` — שומרים כ־`.py` ומריצים `manim -pql … TexLineScene`.
   - **בלוק 3:** אנימציית כתיבה + אם סומן: `Line` + `Create` מתחת לשורה.

## שימוש (לשונית «צירים וגרפים»)

1. מגדירים `Axes` / `NumberPlane`, טווחים, אורכים, `include_tip`, ואופציונלית תבנית עברית.
2. **תזמון צירים** — משך `Create` של הצירים והמתנה אחריו (ברירות מחדל בטופס).
3. (אופציונלי) תחת **אובייקטים על הצירים**: לחצו **+**; בכל שורה — גרף או נקודה, צבע, **משך ניגון** ו**המתנה אחרי**. אופציה **נגן במקביל לשורה הקודמת** (לא בשורה הראשונה): ממזגת את ה־`Create` לשורת `play` אחת. ליד הכותרת **?** — דוגמאות `numpy`. אפשר בלי אובייקטים (רק צירים).
4. **צור קוד** — שלושת הבלוקים מתעדכנים (`run_time` / `wait`; במקביל — `self.play(Create(…), Create(…), …)`).

## הערות

- שם הגופן חייב להתאים לגופן **מותקן במערכת** (XeLaTeX דרך `fontspec`). אם הקומפילציה נכשלת — בדקו התקנה או שם מדויק כפי שמופיע במערכת.
- אם מספר תווי `$` בשורה אינו זוגי, מוצגת אזהרה (כנראה `$` לא סגור).
- `$$` / נוסחאות מורכבות: הכלי מיועד ל־**inline** `$...$` פשוט; ללא תמיכה מובטחת ב־`$$`.
